Pyo is a Python module written in C to help digital signal processing script creation.

* Python 2.5 must be installed on your system before running this installer. *

http://www.python.org/download/releases/2.5.4/

When you install Python, You need to add Python25 folder path to the system variable PATH.
By doing this, the system will accept "python" as an internal command.

Add C:\Python25 to the list of paths in:

system -> Advanced -> Environment Variables -> PATH

This installer will leave a folder called pyo_examples on the Desktop, it's a good 
starting point to explore the library!

In a Command Prompt:

cd Desktop\pyo_examples
python xnoise_example.py
